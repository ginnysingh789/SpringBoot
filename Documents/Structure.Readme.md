# Topic 8 – Project Structure & Best Practices (Packages & Layers)

## Goal

- Organize a Spring Boot project into clear layers:
    - controller
    - service
    - repository
    - model
    - exception
    - config

## Recommended Package Structure

```text
com.example.clean
├─ CleanApplication
├─ controller
├─ service
├─ repository
├─ model
├─ exception
└─ config
